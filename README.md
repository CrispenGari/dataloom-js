### Getting started

First you need to install `dataloom` using your favourite package manager.

```json
{
  "compilerOptions": {
    "strictPropertyInitialization": false
  }
}
```

#### Installation

##### 1. using `npm`

```shell
nmp install dataloom
```

##### 2. using `yarn`

```shell
yarn add dataloom
```

##### 3. using `pnpm`

```shell
pnmp install dataloom
```
